# UserTrack

A professional user management system built with Python and Flask to explore full-stack application development. UserTrack provides secure authentication, user management workflows, database integration, and a responsive interface with light and dark mode support.

## Features

* Secure user authentication and session management
* User registration and login system
* Create, view, update, and delete (CRUD) user records
* Responsive and intuitive dashboard
* Light and dark mode support
* Database integration using SQLite
* Clean and organized project structure

## Tech Stack

* **Backend:** Python, Flask
* **Database:** SQLite
* **Frontend:** HTML, CSS, JavaScript, Bootstrap

## Project Structure

```text
UserTrack/

```

## Getting Started

1. Clone the repository.
2. Create and activate a virtual environment.
3. Install the required dependencies.
4. Initialize the database.
5. Run the application.

## Purpose

This project was built to gain hands-on experience with Python full-stack development and understand how authentication, database management, and responsive user interfaces come together in a complete web application.

## Future Enhancements
